# 🚀 Powercoders Portfolio - Yasha Sharma

This is the repository for my personal portfolio, which showcases the skills and projects I developed during the **Powercoders** program. It serves as a living document of my journey transitioning into the tech industry, highlighting my commitment to clean code, responsive design, and effective problem-solving.

---

## 🎯 Project Goals

The primary goals in building this portfolio were:

1.  **Technical Showcase:** To demonstrate proficiency in fundamental web technologies (HTML, CSS, JavaScript) and key concepts like responsiveness and dynamic user interfaces.
2.  **Professional Presence:** To provide recruiters and collaborators with a centralized, professional overview of my background, soft skills, and contact information.
3.  **Learning Validation:** To solidify my understanding of front-end development principles by applying them in a practical, user-facing project.

---

## ✨ Core Technology Stack

This portfolio is built using a foundational stack to ensure speed, stability, and broad browser compatibility.

| Technology | Purpose |
| :--- | :--- |
| **HTML5** | Provides the semantic structure and content layout. |
| **CSS3** | Responsible for the modern, clean styling and responsive layout (Flexbox/Grid). |
| **JavaScript (ES6+)** | Manages interactive features, including smooth scrolling, the sticky header, and event handling. |

---

## 🛠️ Key Features Implemented

The site includes several modern UX and interactive elements:

* **Fixed Navigation:** A custom-styled header that uses JavaScript and the `IntersectionObserver` API for a smooth transition effect when scrolling.
* **Skill Marquee:** A CSS-driven infinite loop animation to engagingly present technical skills.
* **Project Cards:** Detailed, linkable cards for each major project, including live demos (where available) and links to the source code on GitHub.
* **Accessibility:** Use of semantic HTML and focus on clean structure to improve accessibility and SEO.
